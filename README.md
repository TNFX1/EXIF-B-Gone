\# 🛡️ EXIF-B-Gone



> \*\*Privacy-First Local EXIF Metadata Remover \& Viewer\*\*



EXIF-B-Gone is a modern web and desktop utility designed to view and strip sensitive metadata (GPS location, capture date, device model, ISO settings, etc.) from your images entirely \*\*locally\*\* in your browser.



!\[License](https://img.shields.io/badge/license-MIT-cyan)

!\[Platform](https://img.shields.io/badge/platform-Web%20%7C%20Windows-blue)

!\[Privacy](https://img.shields.io/badge/privacy-100%25%20Client--Side-emerald)



\---



\## ✨ Features



\- 🔍 \*\*EXIF Inspector:\*\* View detailed hidden metadata extracted from your photos.

\- 🧹 \*\*Deep Clean:\*\* Strip location, date, and device information safely at the pixel level.

\- 🔒 \*\*100% Client-Side Privacy:\*\* Your images are \*\*never\*\* uploaded to any server; all processing happens entirely in your local RAM.

\- ⚡ \*\*Fast \& Batch-Ready:\*\* Support for single or batch processing with drag-and-drop.

\- 💻 \*\*Standalone Desktop App:\*\* Pre-built `.exe` releases compiled via GitHub Actions.



\---



\## 🚀 Getting Started



1\. Download the latest `.exe` build from the \*\*Releases\*\* tab to run it directly on Windows.

2\. Alternatively, clone the repository and open `index.html` in any modern web browser.



\---



\## 🛠️ Tech Stack



\- \*\*Frontend:\*\* HTML5, JavaScript (ES6+)

\- \*\*Styling:\*\* Tailwind CSS, FontAwesome

\- \*\*Metadata Parsing:\*\* `exifr` library

\- \*\*Build System:\*\* NW.js \& GitHub Actions Workflow
